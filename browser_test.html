Neo4j Browser Access Test

トラブルシューティング手順

1. 直接アクセスリンク

  • http://localhost:7474/
  • http://127.0.0.1:7474/
  • http://localhost:7474/browser/

2. ブラウザ設定確認

  • JavaScriptが有効になっているか確認
  • アドブロッカーを無効にする
  • プライベートモード/シークレットモードで試す
  • 別のブラウザ（Chrome, Firefox, Edge）で試す

3. キーボードショートカット

  • Ctrl + F5 - ハードリフレッシュ
  • Ctrl + Shift + R - キャッシュクリア
  • F12 - 開発者ツールでエラー確認

4. 接続情報

データベースURL: bolt://localhost:7687
ユーザー名: neo4j
パスワード: testpassword

5. 手動接続

もしブラウザが白い画面のままの場合、以下のURLを手動でコピー&ペーストしてください：

http://localhost:7474/browser/?connectURL=bolt%3A%2F%2Flocalhost%3A7687&username=neo4j
